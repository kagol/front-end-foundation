追一科技于2016年在深圳成立，公司以自然语义理解研究为中心，融合深度学习、机器学习、认知计算等尖端技术，主攻自然语言理解、情感分析及应对，发掘“人工智能”的更多可能并将“赋予机器智慧”的科学研究予以实践。被人工智能权威、创新工场董事长兼CEO李开复评价为国内乃至全球最好NLP公司之一。 追一科技先后获得晨兴资本数百万美元的天使轮投资，高榕资本、晨兴资本数百万美元的A轮融资。并于2017年11月完成2060万美元B轮融资，投资方包括创新工场、GGV、晨兴资本和高榕资本。追一科技团队的技术、产品和商业洞察力获得众多投资者认可。 与此同时，追一科技还获得了众多顶级加速器平台、权威机构和媒体关注。通过加速器平台，与腾讯、万达、微软等探索技术产品、消费场景、企业服务和行业应用等多领域合作。2017年以来，先后获中国好Saas“最具投资价值团队”、“最具价值产品”大奖、“德勤2017中国明日之星”、机器之心Synced Machine Intelligence Awards“全球30大最佳AI创业公司”等荣誉。
